# Profile Card UI 🪪

A clean and responsive profile card built using HTML, CSS, and JavaScript. I built this project to practice UI design, CSS variables, dark mode, and basic DOM manipulation.

## 🚀 Features

- Responsive profile card centered on the page
- Circular avatar with online status indicator
- Name, title, location and short bio
- Skill tags with hover effects
- Animated stats counter that counts up on page load
- Follow button toggle between Follow and Following
- Social media icons for GitHub, LinkedIn, Twitter and Instagram
- Dark mode toggle with smooth transition
- Dark mode preference saved so it remembers on next visit

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, CSS Variables, Transitions)
- JavaScript (DOM Manipulation, localStorage, IntersectionObserver)
- Font Awesome

## ▶️ How to Run

1. Download or clone this repository
2. Open the `profile-card` folder
3. Double click `index.html` to open it in your browser
4. No installation or setup needed

## 📱 Responsive Design

| Screen Size | Behavior |
|---|---|
| Large (900px+) | Wider card, larger text |
| Tablet (600px) | Buttons stack vertically |
| Small Mobile (380px) | Compact padding and font sizes |
